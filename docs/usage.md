Usage:
    git-hist OUTFILE [ -f FILE ] [ -o OPTS ] [ -n ]
    git-hist ( -h | --help )

Options:
    -n, --nostatus          dont check for a clean repo
    -o, --opts OPTS         invoke git log with OPTS
    -f, --file FILE         read log from file
    -h, --help              help for git-hist

Arguments:
    OUTFILE                 write markdown to this file
