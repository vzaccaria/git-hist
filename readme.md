# git-hist
> Generate a HISTORY.md from git log

## Install

Install it with

```
npm install git-hist
```
## Usage

```
Usage:
    git-hist [ -f FILE ] [ -o OPTS ]
    git-hist ( -h | --help )

Options:
    -o, --opts OPTS         invoke git log with OPTS
    -f, --file FILE         read log from file
    -h, --help              help for git-hist

Commands:

Arguments:

```
